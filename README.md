DataGuard Semantic est un moteur intelligent de prévention des fuites de données (DLP). Ce projet utilise les techniques de Traitement du Langage Naturel (NLP) pour analyser automatiquement des textes bruts (emails, tickets, documents) et identifier les contenus critiques avant qu'ils ne soient exposés. Le système détecte trois catégories principales d'informations : les données personnelles (PII), les secrets techniques (mots de passe, clés d'API) et les données métier confidentielles.
